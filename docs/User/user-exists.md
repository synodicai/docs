## GET /\{user\}/exists

Checks if the user exists.

### URL

`GET https://fqk4k22rqc.execute-api.us-east-1.amazonaws.com/{user}/exists`

### Path Parameters

- **\{user\}**: username (string)

### Example Request

```bash
curl -X GET "https://fqk4k22rqc.execute-api.us-east-1.amazonaws.com/{user}/exists"
```
